## Формати файлів

| Формат | Призначення | Рекомендації |
|--------|-------------|--------------|
| **SVG** | Веб, друк | Найкраща якість, масштабується без втрат |
| **PNG @2x** | Веб, презентації | Для екранів з високою роздільністю |
| **PDF** | Друк | Для професійного друку |
| **AI/EPS** | Редагування | За запитом |

## Правила використання

!!! warning "Важливо"
    Завантажуючи та використовуючи ці матеріали, ви погоджуєтесь дотримуватись правил використання бренду, описаних у цьому гайдбуці.

### Дозволено

- Використання для офіційних матеріалів факультету
- Використання партнерами за погодженням
- Використання у навчальних матеріалах УКУ

### Заборонено

- Модифікація логотипів
- Комерційне використання без дозволу
- Використання у контексті, що шкодить репутації
