## Загальні принципи

### Структура макету

Усі корпоративні матеріали базуються на модульній сітці:

- **Поля** — мінімум 15 мм для друку, 40 px для веб
- **Логотип** — завжди у верхньому лівому або правому куті
- **Контактна інформація** — у нижній частині документа

### Колірні схеми

| Схема         | Використання                      |
| ------------- | --------------------------------- |
| **Фіолетова** | Офіційні документи, презентації   |
| **Біла**      | Ділова кореспонденція, бланки     |
| **Змішана**   | Промо-матеріали, соціальні мережі |
