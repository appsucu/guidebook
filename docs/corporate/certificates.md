# Сертифікати

Сертифікати та дипломи підтверджують досягнення учасників програм та курсів факультету.

---

## Типи сертифікатів

<div class="grid cards" markdown>

    ***

- :material-certificate:{ .lg .middle } **Сертифікат програми**

    ***

    Для випускників сертифікатних програм

    ***

## Специфікації

### Формат

| Параметр   | Значення                         |
| ---------- | -------------------------------- |
| **Розмір** | A4 (210 × 297 мм) горизонтальний |
| **Поля**   | 20 мм з усіх боків               |
| **Рамка**  | 3 pt, фіолетовий (#662d91)       |

### Типографіка

| Елемент             | Шрифт               | Розмір |
| ------------------- | ------------------- | ------ |
| **"СЕРТИФІКАТ"**    | Inter ExtraBold     | 28 pt  |
| **Ім'я отримувача** | Inter Medium Italic | 24 pt  |
| **Назва курсу**     | Inter Medium        | 18 pt  |
| **Основний текст**  | Inter Regular       | 12 pt  |

---

## Елементи сертифіката

### Обов'язкові

- Логотип факультету
- Логотип УКУ
- Заголовок "СЕРТИФІКАТ"
- Ім'я та прізвище отримувача
- Назва курсу/програми
- Дата видачі
- Підпис уповноваженої особи

### Опціональні

- QR-код для верифікації
- Номер сертифіката
- Кількість годин/кредитів
- Печатка факультету

## Верифікація

Для забезпечення автентичності сертифікатів рекомендується:

1. **QR-код** — посилання на сторінку верифікації
2. **Унікальний номер** — для перевірки в базі даних
3. **Цифровий підпис** — для електронних версій
