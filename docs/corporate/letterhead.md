Офіційні бланки використовуються для ділової кореспонденції, офіційних листів та документів факультету.

---

## Структура бланку

### Верхній колонтитул

```
┌─────────────────────────────────────────────────┐
│  [Логотип APPS]              [Логотип УКУ]      │
│                                                 │
│  ─────────────────────────────────────────────  │
└─────────────────────────────────────────────────┘
```

### Нижній колонтитул

```
┌─────────────────────────────────────────────────┐
│  ─────────────────────────────────────────────  │
│                                                 │
│  Факультет прикладних наук УКУ                  │
│  вул. Козельницька 2а, кімната 314              │
│  м. Львів, 79026, Україна                       │
│                                                 │
│  +380 32 240 99 44 | apps@ucu.edu.ua            │
│  apps.ucu.edu.ua                                │
└─────────────────────────────────────────────────┘
```

## Специфікації

### Формат

| Параметр       | Значення                                   |
| -------------- | ------------------------------------------ |
| **Розмір**     | A4 (210 × 297 мм)                          |
| **Поля**       | Верх: 25 мм, Низ: 20 мм, Ліво/Право: 25 мм |
| **Орієнтація** | Портретна                                  |

### Типографіка документа

| Елемент                  | Шрифт         | Розмір |
| ------------------------ | ------------- | ------ |
| **Заголовок листа**      | Inter Medium  | 14 pt  |
| **Основний текст**       | Inter Regular | 11 pt  |
| **Контактна інформація** | Inter Regular | 9 pt   |

### Інтервали

| Параметр                  | Значення         |
| ------------------------- | ---------------- |
| **Міжрядковий інтервал**  | 1.5              |
| **Відступ абзацу**        | 0 мм (або 10 мм) |
| **Інтервал між абзацами** | 6 pt             |

---

## Елементи бланку

### Логотип

- Розміщення: верхній лівий кут
- Висота: 15-20 мм
- Версія: повноколірна для світлого фону

### Лінія-розділювач

<div style="height: 3px; background: linear-gradient(90deg, #662d91 0%, #8bc751 100%); margin: 1rem 0; border-radius: 2px;"></div>

- Товщина: 2-3 pt
- Колір: градієнт від фіолетового до зеленого
- Розміщення: під логотипом та над контактами
