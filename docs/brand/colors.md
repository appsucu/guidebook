## Використання кольорів

### Фіолетовий — основний брендовий колір

<div style="background: linear-gradient(135deg, #662d91 0%, #8b5cf6 100%); color: white; padding: 2rem; border-radius: 12px; margin: 1rem 0;">
  <p style="margin: 0; font-size: 1.1rem;">Використовується як головний акцент бренду</p>
</div>

- **Заголовки** — основні назви та титули
- **Фони** — герой-секції, банери, презентації
- **Акценти** — виділення важливих елементів
- **Логотип** — основний колір логотипу

### Зелений — акцентний колір

<div style="background: linear-gradient(135deg, #8bc751 0%, #a3e635 100%); color: #1a1a1a; padding: 2rem; border-radius: 12px; margin: 1rem 0;">
  <p style="margin: 0; font-size: 1.1rem;">Використовується для акцентів та інтерактивних елементів</p>
</div>

- **Крапка в логотипі** — ключовий елемент ідентичності
- **Кнопки** — call-to-action елементи
- **Іконки** — позитивні індикатори, успіх
- **Виділення** — важлива інформація

### Нейтральні кольори

| Колір | Призначення |
|-------|-------------|
| **Темно-сірий** | Основний текст, заголовки на світлому фоні |
| **Сірий** | Другорядний текст, підписи, мета-інформація |
| **Світло-сірий** | Фони секцій, розділювачі, карточки |
| **Білий** | Основний фон, текст на темному фоні |

## Код для розробників

=== "CSS Variables"

    ```css
    :root {
      /* Primary Colors */
      --asf-purple: #662d91;
      --asf-green: #8bc751;
      
      /* Secondary Colors */
      --asf-white: #ffffff;
      --asf-light-gray: #f1f1f2;
      --asf-gray: #9d9fa2;
      --asf-dark-gray: #58595b;
      
      /* Gradients */
      --asf-gradient-purple: linear-gradient(135deg, #662d91 0%, #8b5cf6 100%);
      --asf-gradient-green: linear-gradient(135deg, #8bc751 0%, #a3e635 100%);
    }
    ```

=== "SCSS Variables"

    ```scss
    // Primary Colors
    $asf-purple: #662d91;
    $asf-green: #8bc751;

    // Secondary Colors
    $asf-white: #ffffff;
    $asf-light-gray: #f1f1f2;
    $asf-gray: #9d9fa2;
    $asf-dark-gray: #58595b;

    // Color Map
    $asf-colors: (
      'purple': $asf-purple,
      'green': $asf-green,
      'white': $asf-white,
      'light-gray': $asf-light-gray,
      'gray': $asf-gray,
      'dark-gray': $asf-dark-gray
    );
    ```

=== "Tailwind Config"

    ```javascript
    module.exports = {
      theme: {
        extend: {
          colors: {
            'asf-purple': '#662d91',
            'asf-green': '#8bc751',
            'asf-light-gray': '#f1f1f2',
            'asf-gray': '#9d9fa2',
            'asf-dark-gray': '#58595b',
          }
        }
      }
    }
    ```

## Правила використання

!!! success "Рекомендовано"
    - Використовуйте точні значення кольорів з палітри
    - Забезпечуйте достатній контраст (мінімум 4.5:1 для тексту)
    - Фіолетовий — для основних акцентів, зелений — для другорядних
    - Використовуйте градієнти для динамічних елементів

!!! warning "Уникайте"
    - Не змінюйте відтінки основних кольорів
    - Не використовуйте більше 3 кольорів одночасно
    - Не розміщуйте зелений текст на фіолетовому фоні (і навпаки)
    - Не використовуйте кольори з низьким контрастом для тексту

!!! danger "Заборонено"
    - Заміна брендових кольорів на інші
    - Використання кольорів, що не входять до палітри
    - Зміна прозорості основних кольорів у логотипі
